This is the readme.txt file in the example code for HW02

myStrategy.py:
	The program (which determine the action of "buy" or "sell") submitted by each individual in the class.

rrEstimate.py:
	This script calls myStrategy.py to obtain RR (return rate) for a single stock.
	
rrEstimateAll.py
	This program calls myStrategy.py to obtain the average RR for a given set of 4 stocks. Our judge will use a script similar to this to evaluate your performance.

bestParamByExhaustiveSearch.py:
	Obtain the best set of parameters by exhaustive searchexample

*.csv
	Price data files for 4 different stocks
